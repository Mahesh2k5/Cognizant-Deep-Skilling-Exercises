# Week 3: Spring REST using Spring Boot

There is only one Spring Boot project called `spring-learn` rather than a folder for every single hands-on document.

### Why one project?
The exercise instructions for Week 3 explicitly ask to continuously build upon a single application step-by-step. 

### What's inside?
Instead of starting from scratch every time, I grouped all the sequential tasks directly into this `spring-learn` app:
*   **Hands-on 2 & 4:** Loading XML Beans (`date-format.xml`, `country.xml`) from `src/main/resources`.
*   **Hands-on 3:** Configuring SLF4J logging.
*   **Hands-on 5 & 6:** Demonstrating Prototype vs Singleton scopes and Lists.

You can find all the configurations in `src/main/resources` and the main execution logic inside `SpringLearnApplication.java`.
